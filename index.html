Welcome to GitHub for Google+ Optimizer.

Here you'll find the source for the Google+ Optimizer. Feel free to contribute and add new content. If anyone wants to create an extension for other browsers from this (especially FireFox) that would be great. To make this as easy as possible, all browser-specific methods are used in one single object "Browser.js".

If you just want to use the extension, leave this page and download it directly from google: http://goo.gl/qHYFqt
