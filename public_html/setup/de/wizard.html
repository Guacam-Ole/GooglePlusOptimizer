1.0
wham.png
Filter
Wham - Filter
Zur Weihnachtszeit kann die Dauerberieselung durch “Last Christmas” von Wham schon mal etwas viel werden. Über diesen Filter bleibt Ihr wenigstens bei Google+ davon verschont.
Im Folgenden kannst Du genau einstellen, welche Inhalte gefiltert werden sollen. (Wenn Du nichts filtern möchtest, lass die Einstellungen einfach deaktiviert)

  "WHAM" in Beitrag filtern
Kommt das Wort "WHAM" in einem beliebigen Beitrag vor, so wird dieser automatisch herausgefiltert.

  "WHAM" in Link filtern
Kommt das Wort "WHAM" in einem Link zu einem Beitrag (inkl. Videos) vor, so wird er herausgefiltert.

  "Last Christmas" in Beitrag filtern
Kommen die Wörter "Last Christmas" in einem beliebigen Beitrag vor, so wird dieser automatisch herausgefiltert.

  "Last Christmas" in Link filtern
Kommen die Wörter "Last Christmas" in einem Link zu einem Beitrag (inkl. Videos) vor, so wird er herausgefiltert.

Volltext - Filter
Manchmal reicht ein Hashtag-Filter nicht aus, sondern ein bestimmtes Wort irgendwo im Beitrag reicht, dass man diesen Beitrag nicht lesen möchte. Der Volltextfilter erlaubt genau dies.

Der Volltextfilter benötigt etwas mehr Resourcen als der Hashtag-Filter. Außerdem sollte man beachten, dass man evtl. aus Versehen zu viele Beiträge filter. So filtert der Volltextfilter "Icke" auch Beiträge mit "Zicke" heraus. Hashtags können direkt im Stream gefiltert werden

Bilder - Filter
Wer mehr Wert auf Texte, als auf Bilder legt, kann sie hiermit filtern. Insbesondere bei langsamen Maschinen kann das Filtern von GIFs deutliche Performance-Gewinne bringen. Der Bilderfilter entfernt die Beiträge nicht komplett, sondern blendet einen Hinweis ein, mit dem das jeweilige Bild auf Wunsch angezeigt werden kann.

Dies wird durch folgenden Link dargestellt:

Bild anzeigen ⇘



Erweiterte Einstellungen:
 Nur GIF-Dateien ausblenden
Hierdurch werden ausschließlich GIF-Dateien gefiltert, was 80% der animierten Bilder aus dem Stream entfernt. Animierte JPGs werdem weiterhin dargestellt.

Wetter-Widget
Das Wetter-Widget verhindert, dass Du aus dem Fenster schauen musst… Zusätzlich wird eine Wettervorhersage der nächsten 5 Tage dargestellt. Wie beim Sportwidget betimmt die Auswahl “Positon”, in welcher Spalte das Widget dargestellt werden soll.

Um einen Ort anzugeben, reicht es diesen im Suchfeld einzutragen und auf Suchen zu klicken. Eine Länderangabe, o.ä. ist nicht notwendig. Wird der Ort gefunden, so werden im Auswahlfeld “Ort” die gefundenen Übereinstimmungen dargestellt. Ebenso wie beim Sport müssen diese Einstellungen durch den blauen Button gespeichert werden.

Stoppuhr-Widget
Nie wieder die Pizza anbrennen lassen, wenn Du im Google+ – Stream verloren bist!
Die Stoppuhr zählt die Zeit abwärts, so wie es nunmal eine Stoppuhr üblicherweise macht…

Du stellst Die Zeit in Minuten über den äußeren Ring ein. “Stoppuhr starten” startet dann den Countdown. Der Innere Zeige zeigt die Minuten und am Ende wirst Du durch einen unangenehmen Ton daran erinnert, dass Du die Pizza aus dem Ofen holen wolltest.

2.3
trophies.png
Feature
Trophäen aktivieren
Trophäen sind Belohnungen für Aktionen bei Google+. Auch wenn es keinen echten Preis zu gewinnen gibt, so kannst Du nun auch in Google+ mit Deinen Fähigkeiten angeben. Um die Trophäen zu berechnen müssen Deine sämtlichen (öffentlichen) Beiträge, Kontakte und Profildaten einmalig durchsucht werden.

Die Trophäen werden im "Über mich" - Bereich Deines Profils (nur für Dich) angezeigt.




Für die Nutzung der Trophäen ist eine Authentifizierung erforderlich (warum?)
