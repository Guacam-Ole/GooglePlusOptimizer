Google+ - Filter Einstellungen

Auf der linken Seite gibt es eine Anzeige, welche Filter aktiviert sind. Ist das entsprechende Symbol Schwarz-Weiß, so ist der Filter nicht aktiviert. Zur Aktivierung des jeweiligen Filters diesen einfach anklicken. Er wechselt dann in eine farbige Anzeige.

Beim Überfahren des Symbols mit der Maus wird der Filter ausführlich beschrieben. Die Filter "Hashtag" und "Individuell" müssen zusätzlich über die erweiterten Einstellungen konfiguriert werden.

Sämtliche Einstellungen werden unverzüglich gespeichert
