Soeben wurde eine neue Version des G+O installiert. In dieser Version ging es vor allem darum, das neue Layout von Google+ abzubilden. Ein paar Features haben sich leicht verändert, andere sind entfallen. Teils weil sie nicht mehr notwendig sind, teils, weil ich sie (noch) nicht im neuen Layout umsetzen konnte.

Folgende Änderungen gibt es in dieser Version: Neue Features:
  • Dieser schmucke Updatehinweis :)
Entfallene Features (Dauerhaft):
  • Der InstallationsWizardDas waren mittlerweile einfach zu viele Optionen. Vor allem bei einer Neuinstallation wurde es zu unübersichtlich. Stattdessen jetzt diese Info-Box
  • FußballergebnisseDas lief schon länger nicht mehr ganz rund und wurde auch kaum genutzt
  • QuicksharesSind seit den Collections überflüssig
  • WetterWurde kaum genutzt. Falls das doch noch jemand braucht: Einfach laut meckern :)
  • Bugs & FeaturesAnstelle eines integrierten Bugtrackings jetzt einfach direkter Link zu GitHub
  • Minimieren von Bildern, Videos, etc.
  • YouTube-FilterYouTube-Kommentare tauchen nicht mehr bei Google+ auf.
Temporär entfallene Features (werden nach und nach wieder eingabaut)
  • Trophäen
  • Sprachauswahl
  • Leistungsschutzrecht
  • Hervorheben von Kreislingen
