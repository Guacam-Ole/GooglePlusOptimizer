Neues Update für den Google+ - Optimizer

Soeben wurde eine neue Version des G+O installiert.

Folgende Änderungen gibt es in dieser Version:

Neue Features:
  • Mehr Farben beim Hervorheben von Userbeiträgen
  • Farbliches Hervorheben im "Monopoly-Stil"
  • Hervorheben von LSR-Links
  • Hervorheben von Links zu Seiten, die einen Adblocker erkennen
  • Hervorheben von Links zu Favoriten

Fixes:
  • "English" anstatt "Englisch" im Setup-Menü
