Erweiterte Einstellungen

Auf dieser Seite können erweiterte Einstellungen für einzelne Filter vorgenommen werden. Zur aktivierung der Filter muss zusätzlich auf der Einstellungsseite der jeweilige Filter ausgewählt werden. Die erweiterten Einstellungen werden auch bei Deaktivierung des jeweiligen Filters beibehalten.

Hashtag-Filter

Über den Hashtag-Filter können beliebige Artikel mit vorgegebenen Hashtags herausgefiltert werden.
(Es ist egal, ob die Werte mit oder ohne "#" eingegeben werden)

Individuelle Textfilter

Hierüber lässt sich jeder beliebige Text herausfiltern. Der Filter achtet auf die Groß/Kleinschreibung.
Bitte beachten: Volltextfilter sind sehr arbeitsintensiv und sollten nicht auf schwachen Maschinen benutzt werden.

WHAM-Filter

Über die folgenden Einstellungen lässt sich einstellen, wie "strikt" der WHAM-Filter ausgelegt wird:

  "WHAM" in Beitrag filtern
Kommt das Wort "WHAM" in einem beliebigen Beitrag vor, so wird dieser automatisch herausgefiltert.

  "WHAM" in Link filtern
Kommt das Wort "WHAM" in einem Link zu einem Beitrag (inkl. Videos) vor, so wird er herausgefiltert.

  "Last Christmas" in Beitrag filtern
Kommen die Wörter "Last Christmas" in einem beliebigen Beitrag vor, so wird dieser automatisch herausgefiltert.

  "Last Christmas" in Link filtern
Kommen die Wörter "Last Christmas" in einem Link zu einem Beitrag (inkl. Videos) vor, so wird er herausgefiltert.

Medien-Filter

Über die folgenden Einstellungen lassen sich die Medien-Filter einschränken:

  (Bilder-Filter) Nur GIF-Dateien ausblenden
Hierdurch werden ausschließlich GIF-Dateien gefiltert, was 80% der animierten Bilder aus dem Stream entfernt. Animierte JPGs werdem weiterhin dargestellt.

  (Video-Filter) Nur MP4-Dateien ausblenden
Hierdurch werden ausschließlich Videos im MP4-Format gefiltert, was die meisten animierten Vorschaubilder ("Auto-Awesome") entfernt.

Pause

Jedes Mal, wenn in Google+ bis ans Ende gescrollt wird, wird neuer Inhalt nachgeladen. Dann muss der Filter erneut prüfen und die Seite ggf. anpassen. Um der Seite die Möglichkeit zu geben, den gesamten Inhalt korrekt darzustellen wartet der Filter eine gewisse Zeit, bis er erneut aktiv wird. Solltest Du eine leistungsschwache Maschine besitzen, erhöhe den Wert. Wenn Du anders herum möchtest, dass der Filter schneller reagiert, reduziere ihn.
