Schnapperama

Die Schnapperama - Extension macht nix weiter, als Dir automatisch beim Sparen auf Amazon zu helfen :)

Sobald Du auf einer europäischen Amazon-Seite bist, zeigt Dir die Extension an, ob der gleiche Artikel in einem anderen Amazon-Store günstiger ist. Du kannst die Erweiterung kostenlos direkt im Chrome Webstore herunterladen. Eine umfangreiche Beschreibung zur Extension gibt es bei schnapperama.de.

Gibt es eine Pro-Version?


Nö. Eine Pro-Version gibt es nicht und wird es auch nie geben. Alle Features sind kostenlos.

Wenn Du das Projekt unterstützen willst, darfst Du aber natürlich gerne einen freiwilligen Betrag über PayPal spenden. Dadurch kann ich das Plugin auch zukünftig weiterentwickeln.
