Welcome to the Google+ - Optimizer

The purpose of this optimizer is to remove unwanted content and to add additional features to your Google+ - stream. To achieve this you can deactivate single Google+ - features or just remove specific content.

Enable or disable the single features with the help of the filters. To get more into detail use the settings to modify advanced filter settings.

Additional features can be added in the widgets area. To get more into detail use the settings to modify advanced filter settings.
