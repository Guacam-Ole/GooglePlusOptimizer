Advanced Settings

You can modify your settings for some filters here. To also have to activate each filter on the settings page. The advanced settings will be saved even if you disable the corresponding filter.

Hashtag-Filter

You can filter any hashtag out of the stream.
(It doesn't care if you do or don't enter the '#')

Custom Filter

With these filters you can remove topics with any text included. This filter is case-sensitive.
Please be advised: The custom filter needs a lot of processor power and should not be used on weak machines.

WHAM-Filter

You can decide how "strict" the WHAM-Filter works:

  Filter "WHAM" in post
If "WHAM" appears in any post it will be filtered.

  Filter "WHAM" in Link
If "WHAM" appears in a link it will be filtered.

  Filter "Last Christmas" in Post
If "Last Christmas" appears in any post it will be filtered.

  Filter "Last Christmas" in Link
If "Last Christmas" appears in a link it will be filtered.

Delay

Every time you scroll to the end of a google+ - page, new content will be loaded. Than the filter has to check again if any of its rules apply. This will not happen before a specific period of time has passed to reduce the power and memory needed for the filter. If you own a very old machine you might want to increase this value. On the other side if you got a state-of-the-art personal computer you can reduce this value.
