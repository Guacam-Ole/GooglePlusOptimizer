Nö. Eine Pro-Version gibt es nicht und wird es auch nie geben.

Alle Features bleiben also immer kostenlos.

Hilfe


Wizard-Verhalten:
  • Bei neuer Version anzeigen (Standard)
  • Deaktivieren
  • Wizard zurücksetzen (Alle Features zeigen)


Was eine Einstellung genau macht, erfährst Du, wenn Du im Einstellungsmenü über die Bilder fährst. Ist das Bild grau, so ist die Funktion deaktiviert, ansonsten aktiviert.

Solltest Du weitere Hilfe benötigen oder etwas haken, stell am besten eine Frage in der Support-Community. Verstehe aber bitte, dass ich keinen 24/7 - Support leisten kann.

Es können auch gerne Fehler gemeldet werden oder Verbesserungsvorschläge gemacht werden.

Online ist zudem eine ausführliche Hilfe verfügbar
